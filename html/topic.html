A Brief History on Animated Portable Network Graphics(APNG) -

Created in 2004 by Stuart Parmenter and Vladimir Vukićević of the Mozilla Coropration to store animations needed for interfaces like throbbers(A kind of animation also known as loading icones). the Animated Portable Network Graphics(APNG) is a file format which extends on the PNG file format to play a series of png images. The APNG file format supports 24 bit color as well as 8 bit transparency. The APNG file format as of now is still not as popular as GIF's as alot of web browsers and programs do not support as well as the files being larger then GIF files which is a major downside as the files will hog the users bandwidth.
Some similar file formats include

  • GIF - one of the oldest file formats, it supports 8 bit color and lacks alpha transparency, dispite this it is one of the most popular animated image file formats.
  • WebP - Created very recently in 2010, it boasts features similar to APNG supporting 24 bit color and 8 bit transparency and also features lossy and lossless compression allowing it to have very small file sizes while retaining the highest quality.

APNG File Format -

PNG Signature IHDR (PNG Header) acTL fcTL IDAT (First PNG Image) fcTL fdAT fcTL fdAT IEND (End Block)
Frame 1 Frame 2 Frame 3

  • Animation Control Chunk (acTL) - Indicates that this file is an animated PNG.
  • Frame Control Chunk (fcTL) - Signifies the start of each frame and holds metadata such as resolution, transparency etc..
  • Frame Data Chunk (fdAT) - Stores the frames content.

Main Points about APNG -

  1. Published - 2004
  2. Color Depth - 24 Bit
  3. Frame Rate - Unlimited
  4. Transparency - 8 Bit
