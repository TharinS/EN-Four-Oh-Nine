A Brief History on Animated Portable Network Graphics(APNG)

Created in 2004 by Stuart Parmenter and Vladimir Vukićević the Animated Portable Network Graphics(APNG) is a file format which extends on the PNG file format to play a series of png images similarly to the ever so popular animated GIFs.

Main Points about APNG

  1. Published - 2004
  2. Color Depth - 24 Bit
  3. Frame Rate - Unlimited
  4. Transparency - 8 Bit
